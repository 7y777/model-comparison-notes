# 遥感与目标检测模型对比笔记

本笔记用于记录和对比不同深度学习模型在遥感图像目标检测任务（如水面漂浮物检测、山火烟雾识别等）中的性能表现、结构差异及适用场景。

---

## 1. 核心评估指标定义

为了客观评价各模型的检测精度，我们主要采用以下指标：

* **精确率 (Precision):** 预测为正样本的物体中真正正样本的比例。
* **召回率 (Recall):** 所有真正正样本中被预测出来的比例。
* **交并比 (IoU):** 模型预测出的边界框（Bounding Box）与真实边界框的交集面积与并集面积之比。

$$IoU = \frac{\text{Area of Overlap}}{\text{Area of Union}}$$

* **平均精度均值 (mAP@0.5):** 当 IoU 阈值设为 0.5 时，所有类别的平均精度的平均值。

| 模型名称 | mAP@0.5 (%) | 推理速度 (ms/帧) | 参数量 (M) | 优势亮点 | 局限性 |
| :--- | :---: | :---: | :---: | :--- | :--- |
| **YOLOv8s** | 89.5 | 12.5 | 11.2 | 单阶段检测，速度极快，适合实时监测与部署 | 对密集小目标的定位精度偶有轻微漂移 |

## 3. 典型损失函数 (Loss Function) 结构

在目标检测中，损失函数通常由分类损失（Classification Loss）和定位边界框回归损失（Bounding Box Regression Loss）两部分组成。以常用的 CIoU 损失为例，其考虑了重叠面积、中心点距离以及宽高比：

$$\mathcal{L}_{CIoU} = 1 - IoU + \frac{\rho^2(b, b^{gt})}{c^2} + \alpha \nu$$

> **💡 笔记要点：** > * 在遥感图像处理中（如无人机航拍图），由于目标通常较小且背景复杂（水体、植被、土壤干扰），**YOLOv8** 凭借其 **Anchor-Free（无锚框）** 设计和全新的 **C2f 模块**，在保持高速度的同时，对多尺度目标的捕获能力显著优于前代。

---

## 4. 后续优化方向
1. **轻量化改进：** 尝试引入 GhostNet 或 MobileNet 骨干网络，进一步压缩模型以适配无人机嵌入式设备。
2. **注意力机制：** 计划在特征融合网络（PANet/FPN）中嵌入 SE 或 CBAM 注意力机制，以增强模型对微小水面漂浮物/早期火点的感知能力。
3. **数据增强：** 针对样本不均衡问题，引入 Mosaic4x 及 Mixup 增强策略。
